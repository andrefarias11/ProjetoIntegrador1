Menu

O componente Menu é usado para criar menus e itens de menu em um aplicativo JavaFX. Veja um exemplo de criação e uso de um Menu:
Menu menu = new Menu("Arquivo");
MenuItem item1 = new MenuItem("Abrir");
MenuItem item2 = new MenuItem("Salvar");
menu.getItems().addAll(item1, item2);
É necessário adicionar os itens de menu desejados ao Menu usando o método getItems().addAll().
