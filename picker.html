Picker ou selecionador

Os componentes Picker são usados para permitir que o usuário selecione uma data, arquivo, cor, hora, etc. Existem diferentes tipos disponíveis, como DatePicker, FileChooser e ColorPicker. Aqui está um exemplo de uso do DatePicker:
DatePicker datePicker = new DatePicker();
LocalDate dataSelecionada = datePicker.getValue();
O método getValue() retorna a data selecionada pelo usuário.
