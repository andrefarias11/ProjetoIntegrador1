Animação Fade in e Fade out

A animação de Fade In e Fade Out no JavaFX é usada para criar transições suaves entre elementos visuais, onde um elemento desaparece gradualmente (fade out) e outro elemento aparece gradualmente (fade in). Aqui está uma explicação básica sobre como implementar essas animações:
public class TutorialAnimacaoFadeJavaFX extends Application {
@Override
public void start(Stage primaryStage) {
// Criação do retângulo
Rectangle rectangle = new Rectangle(100, 100, Color.RED);
// Criação da animação Fade In
FadeTransition fadeIn = new FadeTransition(Duration.seconds(2), rectangle);
fadeIn.setFromValue(0.0);
fadeIn.setToValue(1.0);
// Criação da animação Fade Out
FadeTransition fadeOut = new FadeTransition(Duration.seconds(2), rectangle);
fadeOut.setFromValue(1.0);
fadeOut.setToValue(0.0);
// Execução da animação Fade In seguida do Fade Out
fadeIn.setOnFinished(event -> fadeOut.play());
fadeIn.play();
// Criação do layout e adição do retângulo
StackPane root = new StackPane();
root.getChildren().add(rectangle);
// Criação da cena com o layout como raiz
Scene scene = new Scene(root, 200, 200);
// Configuração da cena no palco principal (Stage)
primaryStage.setScene(scene);
primaryStage.setTitle("Exemplo de Animação Fade JavaFX");
primaryStage.show();
}
public static void main(String[] args) {
launch(args);
}
}
