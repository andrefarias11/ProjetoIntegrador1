Tabela com reordenação das linhas da tabela, ao clicar no título da coluna:

O componente TableView é usado para exibir dados em forma de tabela em um aplicativo JavaFX. Veja um exemplo de criação e uso de um TableView:
TableView tableView = new TableView<>();
TableColumn, String coluna1 = new TableColumn<>("Coluna 1");
TableColumn, Integer coluna2 = new TableColumn<>("Coluna 2");
tableView.getColumns().addAll(coluna1, coluna2);
É necessário adicionar as colunas desejadas ao TableView usando o método getColumns().addAll(). Para permitir a reordenação das linhas, você precisará implementar a lógica apropriada.
