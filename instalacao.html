Tutorial através da documentação do JavaFX

Instalação no Windows

Certifique-se de ter o Java Development Kit (JDK) instalado no seu computador. Você pode baixá-lo no site oficial da Oracle

Após instalar o JDK, verifique se a variável de ambiente JAVA_HOME está configurada corretamente. Para fazer isso, abra o prompt de comando e digite java -version. Se exibir a versão do JDK instalada, significa que a variável de ambiente está configurada corretamente.

Acesse o site oficial do JavaFX e clique em "Downloads". Baixe a versão do JavaFX correspondente à versão do JDK instalada. Por exemplo, se você instalou o JDK 11, baixe o JavaFX 11.

Extraia o arquivo baixado para um diretório de sua escolha.

Agora, é necessário configurar o JavaFX no ambiente de desenvolvimento. Abra o IDE (Eclipse, IntelliJ, NetBeans, etc.) que você está utilizando para desenvolver suas aplicações JavaFX.

No IDE, encontre as configurações do projeto (geralmente nas propriedades do projeto ou nas configurações do Build Path).

Adicione a biblioteca do JavaFX ao classpath do projeto. Selecione a opção "Add External JARs" (ou similar) e navegue até o diretório onde você extraiu o arquivo do JavaFX. Selecione todos os arquivos JAR disponíveis e adicione-os ao projeto.

Agora, você está pronto para começar a desenvolver com o JavaFX no Windows!

Lembre-se de que essas instruções podem variar dependendo do IDE que você está usando. Verifique a documentação do seu IDE específico para obter instruções detalhadas sobre a configuração do JavaFX.
