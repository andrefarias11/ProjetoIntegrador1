Barra de Progresso

O componente ProgressBar é usado para exibir o progresso de uma tarefa em um aplicativo JavaFX. Veja um exemplo de criação e uso de uma ProgressBar:
ProgressBar progressBar = new ProgressBar();
progressBar.setProgress(0.5);
O método setProgress() define o valor do progresso da barra, que varia de 0 a 1.
