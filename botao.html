Botão

O componente Button é usado para criar botões interativos em um aplicativo JavaFX. Veja um exemplo de criação e configuração de um botão:
Button button = new Button("Clique aqui");
button.setOnAction(event -> {
// Lógica a ser executada quando o botão for clicado
System.out.println("Botão clicado!");
});
O método setOnAction() permite definir uma ação a ser executada quando o botão for clicado.
