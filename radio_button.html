Radio button

O componente Radio Button é usado para permitir que o usuário selecione uma única opção de um conjunto de opções mutuamente exclusivas. Veja um exemplo de criação e uso de Radio Buttons:

ToggleGroup group = new ToggleGroup();
RadioButton radioButton1 = new RadioButton("Opção 1");
radioButton1.setToggleGroup(group);
RadioButton radioButton2 = new RadioButton("Opção 2");
radioButton2.setToggleGroup(group);
O uso do ToggleGroup garante que apenas uma opção de RadioButton dentro do mesmo grupo possa ser selecionada
