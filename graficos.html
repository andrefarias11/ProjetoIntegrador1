Graficos

No JavaFX, você pode criar gráficos e visualizações interativas usando a classe Chart e suas subclasses. O JavaFX fornece várias subclasses de Chart, como LineChart, BarChart, PieChart, entre outras, para atender diferentes tipos de gráficos. Aqui está uma explicação básica sobre como criar gráficos em JavaFX:
public class TutorialGraficoJavaFX extends Application {
@Override
public void start(Stage primaryStage) {
// Criação dos eixos do gráfico
CategoryAxis xAxis = new CategoryAxis();
NumberAxis yAxis = new NumberAxis();
// Criação do gráfico de barras
BarChart, Number barChart = new BarChart<>(xAxis, yAxis);
barChart.setTitle("Exemplo de Gráfico de Barras");
xAxis.setLabel("Categorias");
yAxis.setLabel("Valores");
// Criação da série de dados
XYChart.Series, Number series = new XYChart.Series<>();
series.getData().add(new XYChart.Data<>("Categoria 1", 10));
series.getData().add(new XYChart.Data<>("Categoria 2", 20));
series.getData().add(new XYChart.Data<>("Categoria 3", 15));
// Adição da série ao gráfico
barChart.getData().add(series);
// Criação da cena com o gráfico como raiz
Scene scene = new Scene(barChart, 400, 300);
// Configuração da cena no palco principal (Stage)
primaryStage.setScene(scene);
primaryStage.setTitle("Exemplo de Gráfico JavaFX");
primaryStage.show();
}
public static void main(String[] args) {
launch(args);
}
}
