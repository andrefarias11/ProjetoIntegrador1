Toggle

O componente Toggle é usado para permitir que o usuário alterne entre dois estados, como "ligado" e "desligado". Aqui está um exemplo de criação e uso de um Toggle:
ToggleButton toggleButton = new ToggleButton("Ligar/Desligar");
toggleButton.setOnAction(event -> {
boolean estado = toggleButton.isSelected();
// Lógica a ser executada quando o ToggleButton for alterado
System.out.println("Estado: " + estado);
});
O método setOnAction() permite definir uma ação a ser executada quando o estado do ToggleButton for alterado.
