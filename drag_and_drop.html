Drag and Drop

A funcionalidade de Drag and Drop permite arrastar elementos de um local para outro em um aplicativo JavaFX. Veja um exemplo básico de como habilitar o Drag and Drop:
Node node = ...; // Nó que deseja tornar "arrastável"
node.setOnDragDetected(event -> {
Dragboard dragboard = node.startDragAndDrop(TransferMode.MOVE);
ClipboardContent content = new ClipboardContent();
content.putString("Informações adicionais");
dragboard.setContent(content);
});
Node targetNode = ...; // Nó que receberá o elemento arrastado
targetNode.setOnDragOver(event -> {
if (event.getGestureSource() != targetNode && event.getDragboard().hasString()) {
event.acceptTransferModes(TransferMode.MOVE);
}
event.consume();
});
targetNode.setOnDragDropped(event -> {
Dragboard dragboard = event.getDragboard();
boolean sucesso = false;
if (dragboard.hasString()) {
// Lógica para processar o elemento arrastado
sucesso = true;
}
event.setDropCompleted(sucesso);
event.consume();
});
Nesse exemplo, o "node" é torna "arrastável" e o "targetNode" é o destino onde o elemento arrastado pode ser solto.
