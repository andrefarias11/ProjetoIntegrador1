Tabs

O componente TabPane é usado para exibir várias guias em um aplicativo JavaFX, permitindo que o usuário alterne entre elas. Veja um exemplo de criação e uso de um TabPane:
TabPane tabPane = new TabPane();
Tab tab1 = new Tab("Guia 1");
Tab tab2 = new Tab("Guia 2");
tabPane.getTabs().addAll(tab1, tab2);
É necessário adicionar as guias desejadas ao TabPane usando o método getTabs().addAll().
