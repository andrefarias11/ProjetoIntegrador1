Slider

O componente Slider é usado para permitir que o usuário selecione um valor em um intervalo contínuo. Veja um exemplo de criação e uso de um Slider:
Slider slider = new Slider(0, 100, 50);
slider.valueProperty().addListener((observable, oldValue, newValue) -> {
double valorSelecionado = slider.getValue();
// Lógica a ser executada quando o valor do Slider for alterado
System.out.println("Novo valor: " + valorSelecionado);
});
O método valueProperty().addListener() é usado para detectar alterações no valor selecionado pelo usuário.
