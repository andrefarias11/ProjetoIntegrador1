Input_fields

Os componentes de Input Fields são usados para receber entrada de texto do usuário. Existem diferentes tipos disponíveis, como TextField, PasswordField e TextArea. Aqui está um exemplo de uso do TextField:
TextField textField = new TextField();
String texto = textField.getText();
O método getText() retorna o texto digitado pelo usuário no campo de texto.
