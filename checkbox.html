Checkbox

O componente Checkbox é usado para permitir que o usuário selecione uma ou mais opções. Aqui está um exemplo de criação e uso de um Checkbox:
CheckBox checkBox = new CheckBox("Opção 1");
boolean selecionado = checkBox.isSelected(); O método isSelected() retorna true se o Checkbox estiver selecionado e false caso contrário.
